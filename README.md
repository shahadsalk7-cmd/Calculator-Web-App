# Calculator Web Application

A functional web-based calculator built using HTML.

## Overview
This project demonstrates the implementation of core arithmetic operations within a clean and structured user interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Simple and user-friendly layout

## Purpose
The goal of this project was to strengthen front-end fundamentals and apply logical problem-solving skills in building an interactive web application.

## Technologies Used
- HTML
